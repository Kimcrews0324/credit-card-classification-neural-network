# Credit Card Usage Classification Using a Neural Network

## Overview

This project demonstrates the implementation of a simple feedforward neural network using Python and NumPy to classify credit card usage behavior. The model predicts whether an individual uses a credit card based on demographic and financial attributes.

The project focuses on core machine learning concepts including data preprocessing, neural network architecture, forward propagation, classification, and performance evaluation.

## Objective

Develop a basic neural network capable of classifying credit card usage using input features such as years of experience and salary.
